void DG::legendre ( int  p,
Scalar  x,
Scalar &  L0,
Scalar &  L0_1,
Scalar &  L0_2 
)

Compute the orthogonal legendre polynomial and its first & second derivatives, given p and x They are computed using the recurrence relation phi_0(x) = 1 -> 0th degree legendre polynomial phi_1(x) = x -> 1st degree ... phi_n(x) = a * x * phi_{n-1}(x) - b * phi_{n-2}(x) where a = (2n - 1) / n b = (n - 1) /n The first and second derivative can be obtained similarily applying product rule of derivaties on the recurrence relation.
